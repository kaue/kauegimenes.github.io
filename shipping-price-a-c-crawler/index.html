Shipping Price - A C# Crawler

Shipping Price shows real time information about shipping price for diferent online retailers, with this tool they are able to check and ajust the shipping price policy to the market.

Code

This project was divided in two parts:
The crawler that gets data of multiple websites and generate a xml output, its developed using C# and is scheduled to run multiple times during the day, to recive data and update the dashboard.
The Dashboard was developed using PHP and its integrated with the C# using a XML file, the role of this part is to show all the information to use user.

Why

The main goal of this project is to offer real time information about shipping price. By using real time data, online retailers can check if its shipping price policy is compatible with the market.
